# Timesheet Local Context

Personal configuration for the timesheet skill. Copy this file to `.local-context.md` and customize it.

## Excluded projects

Projects listed here will never have Harvest time entries created. Use this for personal or non-billable work that appears in your daily logs.

- my-side-project

## Internal project

The Harvest project name used for the daily 1-hour admin entry and as a fallback for unmatched log entries.

```
Acme Corp [INT] Internal
```

## Admin task name

The task name on the internal project used for the daily admin entry.

```
Admin
```
