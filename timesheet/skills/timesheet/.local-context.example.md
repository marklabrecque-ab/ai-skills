# Timesheet Local Context — moved

As of the per-ticket / sidecar refactor, this skill no longer reads its config from inside the skill directory. Configuration now lives in **`~/daily_reports/meta/`** and is shared with other skills (e.g. `value-estimates`).

- **`~/daily_reports/meta/projects.yml`** — project map (Harvest ↔ GitLab ↔ log aliases), excluded projects, internal project + admin task.
- **`~/daily_reports/meta/local-context.md`** — per-machine settings (daily reports path).
- **`~/daily_reports/meta/tickets.json`** — sidecar rollup written by this skill; do not edit by hand.

See `SKILL.md` Step 0 for the schema details. The legacy `.local-context.md` in this directory is no longer read.
